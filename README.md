# My First Quiz App 🎯

This is my very first web development project. I created this simple and interactive Quiz Application using HTML, CSS, and JavaScript.

## Live Demo

👉 [Play Quiz Now] (https://ajaykushwaha214.github.io/html-css-quiz-app/)

## About the Project 

I have recently started learning web development. This quiz app contains multiple-choice questions mainly focused on HTML and CSS. 

I built this project to practice and test what I have learned so far. It helped me understand how HTML, CSS, and JavaScript work together.

## Features

- Clean and attractive design
- Multiple choice questions with 4 options
- Instant score calculation at the end
- Fully responsive (works well on mobile and desktop)
- Fun wrong answer options

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Download or clone this repository
2. Go inside the project folder
3. Open the index.html file in any web browser (Chrome, Firefox, etc.)
4. Start playing the quiz!

## What I Learned From This Project

- How to create a proper webpage structure using HTML
- Styling the page beautifully with CSS
- Making the quiz interactive using JavaScript
- Working with arrays and DOM manipulation
- Connecting HTML, CSS, and JavaScript files together

## Future Plans
